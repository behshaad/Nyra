# Atomic Course Releases for Learner-Visible Structure

Course Structure and Journey Presentation remain independent revision aggregates, but compatible reviewed revisions are promoted together as one atomic Course Release whenever a structural change affects the learner-visible Practice Journey. This prevents Published hierarchy from referencing missing presentation or Published presentation from relying on incompatible structure without coupling ordinary content authoring to visual configuration.

A Course Structure Revision may publish independently only when it cannot change learner-visible hierarchy, such as adding an empty or Draft-only container. A Journey Presentation Revision may publish independently only when it validates against the current Published Course Structure. Release validation identifies required companion revisions, and learner rendering does not invent fallback coordinates to hide an incomplete release.
