# B1 first rollout question types

Nyra's first B1 rollout will use the existing multiple choice, fill-in-blank, and word ordering Question Types rather than adding long-writing, free-speaking, or subjective scoring interactions. B1 difficulty will come from richer original prompts, longer travel and announcement-style comprehension, stronger distractors, past-tense narration, and more complex grammar targets while keeping evaluation inside the proven deterministic Question engine.
