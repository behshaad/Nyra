# Layered learner Flashcards

The learner-facing Flashcards experience will use guided layers inside `/flashcards` instead of a single dense page or modal-based creation. Home remains a library view with ready-made and personal Flashcard Decks, major layer state is represented with query parameters for natural refresh/back behavior, and deck study plus personal deck creation/editing happen in focused full-content layers. This preserves the existing route and domain model while making Flashcards feel calmer, more guided, and easier to extend.
