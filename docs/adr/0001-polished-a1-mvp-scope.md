# Polished A1 MVP scope

Nyra will launch its first real MVP around a polished A1 learning path instead of a broad A1-B2 path. This keeps the first release focused on proving the core learning loop, admin-authored content quality, progress, flashcards, and trust before expanding into higher levels.

## Consequences

A1-B2 remains the long-term content ambition, but the product must not imply that all levels are equally complete at launch. Higher levels can appear as future roadmap or locked coming-soon areas only when that helps learners understand where Nyra is going.
