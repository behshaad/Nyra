# Optional flashcard pronunciation audio

Flashcards may include optional Pronunciation Audio as a card asset. This is a scoped exception to the A1 text-only default: audio can support individual Flashcards, but it does not introduce speech evaluation, required audio completion, or an audio-first learning path.

For the current phase, Nyra references Pronunciation Audio by site path or trusted external URL rather than owning upload and storage. This keeps the Flashcard study experience closer to the learner's pronunciation need while preserving the text-only A1 delivery path described in ADR 0014.
