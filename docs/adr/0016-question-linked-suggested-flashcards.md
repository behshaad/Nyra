# Question-linked Suggested Flashcards

Nyra will show authored Suggested Flashcards after any incorrect Question Attempt, including regular Skills, Unit Checkpoints, and the Final A1 Test. This keeps remediation tied to the specific learning gap revealed by the attempt while preserving Soft Gates: assessments can recommend review without blocking continued progress.

Admins will attach Suggested Flashcards from Published admin-authored Flashcard Decks only, so learner feedback does not expose learner-created, Draft, In Review, or Archived cards.

Incorrect-answer feedback will show Suggested Flashcards as lightweight previews with a path into the Flashcards experience, while actual Spaced Repetition review remains in Flashcards. This avoids treating a quick remediation glance during a Question Attempt as a scheduled Flashcard review.

If an attached Suggested Flashcard later belongs to a deck that is no longer Published, learner feedback will hide it while admin screens keep the link visible with an availability warning.
