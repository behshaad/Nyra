# A2 question quality blueprint

Nyra's full A2 implementation will use richer Question Type interactions and a consistent authored quality blueprint. Regular A2 Skills will use eight Questions with a mix of multiple choice, fill-in-blank, word ordering, and applied comprehension; Unit Checkpoints will use twelve scoreable Questions across the Unit; the Final Level Test will use forty scoreable Questions with a 75% passing score. Each A2 Unit will also receive a published Unit Flashcard Deck and at least one Persian-first Learning Guide, with Suggested Flashcards linked selectively to high-value remediation gaps.

Implementation will happen in reviewable phases: first make the existing three Question Types truly distinct in the learner UI and evaluation engine, then author and publish A2 content in Units 1-4, Units 5-8, and Units 9-12 plus the Final Level Test, with regression tests and seed verification at each stop.
